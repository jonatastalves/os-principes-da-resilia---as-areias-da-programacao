Groudon é uma espécie gigantesca e seu corpo é extremamente resistente a todo e qualquer tipo de ataque. Esse monstro colossal é na verdade areia pura com rochas de montanhas e a única forma de derrota-lo é atingir o seu ponto fraco , através do desafio abaixo.

Escolha a opção abaixo que representa a resposta para a questão: o que é o < !DOCTYPE html > ?

1 - é uma declaração para informar ao navegador qual é a extensão do arquivo utilizado

2 - é uma declaração para informar ao navegador qual é a versão do HTML utilizada.
